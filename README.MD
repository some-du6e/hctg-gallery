## Hosting
1. Install requirements
```bash
pip install -r requirements.txt
```

2. Fill out .env
```env
# automatically generated by import_cookies.py
session_token=
signed_user= # or you can get them from your browser's cookies for auth.hackclub.com
# Slack ###############
SLACK_BOT_TOKEN= # bot token, begins with "xoxb-"
SLACK_APP_TOKEN= # app token, begins with "xapp-"
SLACK_ADMIN= # your Slack user ID, begins with "U"
SLACK_LOOKUP_CHANNEL= # channel ID for looking up projects, begins with "C"
# S3 ###############
CF_ACCOUNT_ID= # Cloudflare account ID, found in the dashboard
CF_ACCESS_KEY_ID= # Cloudflare API key, found in the dashboard
CF_SECRET_ACCESS_KEY= # Cloudflare secret access key, found in the dashboard
# Extra ############
HCTG_BASE_URL= # default is https://game.hackclub.com
IMG_BASE_URL= # public URL for your S3 bucket, e.g. https://r2.hctg.gallery.karimeltaib.com
```

3. Run it 
```bash
python app.py
```
